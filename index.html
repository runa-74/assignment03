1.Why we use selection class?

2.What is dd?

3.How to use rowspan?

ANSWER-

1.We need to define "selection" class in style tag or css file. There is no effect if we just put class="selection"in html file only. we can use the classes to apply CSS to a given element/elements.

2.dd is a html tag.The dd tag in HTML stands for definition description and is used to denote the description or definition of an item in a description list.

3.The rowspan attribute specifies the number of rows a cell should span.

4.The colspan attribute defines the number of columns a cell should span.
